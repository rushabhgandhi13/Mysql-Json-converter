### To run this project :

Clone the project -
```
  $ git clone https://github.com/rushabhgandhi13/Mysql-Json-converter.git
```
  
Install all the reqirements -
```
  $ pip install -r requirements.txt
 ``` 

Run the following commands -

```
python main.py
```
